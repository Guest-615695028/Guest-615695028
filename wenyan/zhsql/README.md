# 中文結構查詢語言
結構查詢語言（Structured Query Language, SQL）始於一九七四年；能多而用簡，人盡可用。1980年10月，美國以爲標準，萬國咸用。
結構查詢語言有三操焉：一曰定，二曰纂，三曰運。其基關係代數，其致明所爲而不明所以爲；而今者基英不基漢，不便國人，多語達一意，吾爲之痛恨嘆惋。
