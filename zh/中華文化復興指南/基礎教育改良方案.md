# 基礎教育改良方案
## 反對唯應試教育
1989年一項應試教學研究確定了七個點，依道德降序爲：
1. 說明考試總目標；
2. 指導一般考試技巧；
3. 指導標準化考試技巧；
4. 針對考試具體題型指導；
5. 以相同格式考卷舉例指導考試具體題型；
6. 要求學生做往年考試卷或做與考試格式、內容相似的模擬卷；
7. 在考試之前數日或期間說明考試具體內容。

該研究得出的結論是：道德界限落在第三點和第五點之間，第一點和第二點是道德的，第六點和第七點是不道德的。[唐納·坎貝爾](https://en.wikipedia.org/wiki/Campbell's_law)說：「一個社會指標越用于社會決策，越容易受腐化壓力影響，也越傾向扭曲和損害其所測定的社會過程。」所以，用于社會決策的指標，需要定期或不定期增減調整比例。

### 義務教育考試方法成績呈現
1. 小學一至二年级無紙筆考試。
2. 小學三至六（五）年級由學校每學期組織一次期末考試，紙筆考試限于語文、數學二門學科，日期爲學期最後一個星期五；三、四年級每學科限時不超過90分鐘，五、六年級每學科限時不超過120分鐘。不組織其他考試。
3. 初中階段從學科實際出發，每學期可適當安排一次期中考試，或不安排之。
4. 僅紙筆考試科目呈現分數、等級（優/A、良/B、合格/C、不合格/F），其他科目結果以「合格/A」、「不合格/F」（F）呈現。

### 小學
1. 增設綜合性課程，重視創造力培養。

### 初中
1. 地理分成兩門學科：天文地理（自然地理、地球科學）和人文地理，高中亦然；
2. 城鎮地區義務教育六年級（無論六三、五四制）下學期可開設一次英語期末考試，聽説爲主、讀寫爲輔（不考察翻譯），爲外語分流依據：僅排名前四分之一至二分之一（具體分流比由學校設定，以三分之一爲宜）者可選擇繼續在中學階段修讀英語，否則强制變更外語語種，具體語種由學生自選，各外語比例由母語人口多寡決定。
3. 恢復初中「歷史與社會」（歷史、人文地理）、「科學」（物理、化學、生物學、天文地理）兩組合并課程，減輕學生負擔。
4. 增大中學非英外語（日、俄、德、法、西班牙語）比例以滿足學生愛好、特長，而不是盲目「廢除英語」，或廢除其他任一外語科目；單一語種班級數、人數不超過總班級數、人數（含英語班）二分之一，人數最多的外語語種在英語、俄語、法語共三種語言中選擇，概率分別爲50%、25%、25%（英語教材需同時提供英式和美式音標，重音標在單詞上；派生詞只標重音，類似俄、德語教材方法）。

### 高中
1. 普通高中在二年級分流爲普通高中和技術高中（藝術、體育、信息技術和通用技術）。
2. 禁止高三年級提前一個月開學（高三年級常見提前至八月一日開學之行爲，因爲高考提前一個月結束下學期），開學時間與高中非畢業年級同日；禁止高三年級師生占用雙休日（社會工作人員調班日除外）、節假日、假期上課，作息原則上與高中非畢業年級統一。
   - 本人上高中所在學校，當時就有高三占用周日、調休日及8月份上課的習慣，僅國際班（已分化）無此行爲。
3. 合并春夏季高考，漢地十八省（河北、河南、山東、山西、陝西、甘肅，江蘇、浙江、安徽、江西、湖北、湖南、四川、福建、廣東、廣西、雲南、貴州）四市（北京、天津、上海、重慶）高中，高三上下學期各有一次選考科目（物理、化學、生物學、天文地理、思想政治、歷史、人文地理）及外語考試機會，上學期在期末考試日，下學期在5月1日前三日（4月28-30日）。若兩次都報考，取高分進入最終成績。
4. 高考前移至5月1日前最後一個工作日舉行，減輕備考生時間壓力，帶動五一消費，一舉兩得。
5. 高三下學期授課以中大銜接而不是備戰高考爲主，畢業典禮于下學期結束當日（與高中非畢業年級散學典禮同時）而非高考完畢當日或次日舉行，此時高考完畢後仍要修讀國家高中選修課程，結合本地知名大學先修成分，以此保障選修課程授課質量，減少不適應大學學習生活的人數。

## 注釋
* 「非英外語」在國内常稱爲「小語種」，但外國或國際語境中兩者不等價：
  * 「小語種」不包含漢語；
  * 英語國家不適用「非英外語」概念，只説「外語」；
  * 「非通用語種」泛指不屬於聯合國六種通用語（漢、英、西班牙、阿拉伯、俄、法語）及其方言的語言。
