# 中華文化復興指南
## 中國式現代化
中國式現代化，習近平謂之「中國共產黨領導的社會主義現代化」，必「居天下之廣居，立天下之正位，行天下之大道。」何謂也？

天下之廣居者，萬物和諧共生之地球，居之則必建人類命運之共同體，保人與自然之和諧共生。\
天下之正位著，中國共產黨之全面絕對領導，立之則必以人民爲中心，促全過程人民民主，豐富人民精神世界，創造人類文明新形態。\
天下之大道者，中國特色社會主義之和平發展道路，必全面深化改革開放、發揚鬥爭精神，以日益高質量發展，乃至十四億人共同富裕而不止。
