# 數據定義
[參見](https://dev.mysql.com/doc/refman/8.0/en/sql-data-definition-statements.html)

## 1 建（CREATE）
有而欲復建，**若無**（IF NOT EXISTS）二字明則不作，不明則覆之。

### 1.1 數據庫（DATABASE）
```
建庫「庫名」[若無]。
```

### 1.2 事務（EVENT）
```
建事「事名」[若無]於<時刻>[每<時間>[至<時刻>]]。
      <語句>
事畢。
時刻：今|<數>年<數>月<數>日<數>時<數>分<數>秒
時間：[<〇至四十八>日][<〇至二十三>時][<〇至五十九>分][<〇至五十九>秒[<〇〇〇至九九九>]]
```
- 時刻：所於不先于今，所至不先于所於。
- 時間：日時分秒，無者置零，不少於一秒。不過四十九日者，[`整一無`](../README.md)存不過四十九日十七時二分四十七秒二九五。

### 1.3 函數（FUNCTION/PROCEDURE）
```
建術「術名」（{入|出}「形參」，...）。
      <語句>
術畢。
建算「術名」（「形參」，...）。
      <語句>
術畢。
```

### 1.4 索引（INDEX）
廢。表以行索。

### 1.5 記錄（LOGFILE GROUP）
廢。[太史](./log.md)書之，不置別史。

### 1.6 服務器（SERVER）
廢。所謂「服務器」者，[聯構](https://dev.mysql.com/doc/refman/8.0/en/federated-storage-engine.html)獨有以至他機，今以前端至之。

### 1.7 空間參考系（SPATIAL REFERENCE SYSTEM）
廢。空間以前端釋。

### 1.8 表（TABLE）
表必有首，首以行（音航）序，自一而增，無則入焉。故`行`字不可以爲列名。

```
建表[若無]「表名」（「列名」<列義>，...[，限（式）]）。
建表[若無]「表名」如「表名」。
建表[若無]「表名」<取語句>。
列義：〈品類〉[置（式）][隱][獨]
      [依「表名」[刪{從|逆|復}][更{從|逆|復}]]。
```

- [彼](https://dev.mysql.com/doc/refman/8.0/en/create-table.html)以限（CONSTRAINT）字明主（PRIMARY）、獨（UNIQUE）、外鍵（FOREIGN KEY），徒增文長，吾不用也。
- `〈品類〉`視[此](../guide.md)。
- 隱：若非既明，`取`表不顯。
- 獨：非所置不得重複。
- 依不逾庫。 - 從：所依刪則刪之，所依改則改之。 - 逆：所依或不得刪，或不得改。 - 復：所依或刪或改則復於零。
- 取語句：[以「取」字始](./dml.md)。

### 表組（TABLESPACE）
廢。

### 觸發器（TRIGGER）

```
建法「法名」[若無]於表[先|後][增|刪|更]。
      <語句>
事畢。
```

隨動不作

### 視圖（VIEW）
視圖引表以作，可[`取`](./dml.md)，不可`增刪更`。
```
建引「視圖名」<取語句>。
```

## 去（DROP）

無而去之，告。

```
去庫「庫名」。
去事「事名」。
去術「術名」。
去表「術名」。
去法「法名」。
去引「引名」。
```

去庫，則其諸表、事、術皆去。

## 改（ALTER）

無而改之，告。

### 數據庫（DATABASE）

改必重建。

### 事務（EVENT）

```
改事「事名」[於<時刻>][每<時間>][至<時刻>]。
時刻：今|<數>年<數>月<數>日<數>時<數>分<數>秒
時間：[<數>日][<數>時][<數>分][<數>秒]
```

- 時刻：所於不先于今，所至不先于所於。
- 時間：日時分秒，無者置零。

### 函數（FUNCTION/PROCEDURE）

改必重建。

### 表（TABLE）

```
改表「表名」<方法>。
方法:
  |建 （「列名」<列義>，...）[先「列名」|末]
  |列「列名」〈品類〉[置（式）][隱|顯][獨|去獨]
      [依「表名」[刪{從|逆|復}][更{從|逆|復}]|去依]。
  |列「列名」名「新列名」
  |去列「列名」
  |限（式）
  |去限
  |名「新表名」。
列義：〈品類〉[置（式）][隱][獨]
      [依「表名」[刪{從|逆|復}][更{從|逆|復}]]。
```

表文不動。
