# 數據操控
[參見](https://dev.mysql.com/doc/refman/8.0/en/sql-data-manipulation-statements.html)
## 增（INSERT）
```
增「表名」列（「列名」，...）置（式組，...）
```
`式組`：式數視列數。
### 例
```
建表「龍虎榜」（「姓」文，「名」文）。
增「龍虎榜」列（「姓」，「名」）置（『趙』，『公明』）（『錢』，『偉昌』）（『孫』，『文』）（『李』，『德』）
```

表`龍虎榜`如是。
|行|姓|名
|-|-|-
|一|趙|公明
|二|錢|偉昌
|三|孫|文
|四|李|德
## 刪 (DELETE)
```
刪「表名」若（條件）計（正整數）。
```
## 取 (SELECT)
```
取（式［名「新列名」］，...）於「表名」［若（條件）］［組以｛「列名」｜（式）｝［使（條件）］］［序以｛「列名」｜（式）｝］。
```
## 表（TABLE）
```
表「表名」。
```
若
```
取各於「表名」。
```
