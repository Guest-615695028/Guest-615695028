# 統一文言編程語言

吾文言編程語言，一字以達意。

[彼](https://github.com/wenyan-lang/wiki/blob/master/Syntax-Cheatsheet.md)云：`吾有一數。曰三。名之曰「甲」。`\
我作：`整甲置三。`

彼云：`吾嘗觀「「算經」」之書。方悟「正弦」「餘弦」「圓周率」之義。`\
我作：`引《算經》得「正弦」「餘弦」「圓周率」。`\
註曰：`引《算經》得一`，「算經」諸法盡可用也。

彼云：`吾有一物。名之曰「甲」。其物如是。物之「「乙」」者。數曰三。物之「「丙」」者。言曰「「丁」」。是謂「甲」之物也。`\
我作：`物〈如是〉有數乙。言丙。物成。〈如是〉甲置「乙曰三，丙曰『丁』」。`

彼之法，蓋譯歐洲計製協會之編碼也，非中國人之法也。

## 關鍵字

```
數字：〇一二三四五六七八九十百千万亿兆京垓升降
品類：爻數整實複列物類術入成引得
控制：若則否也恆爲云凡中至止次歸
運算：置用夫者之進退正負乘除模加減不大小等於在與抑或非
調試：書
```

右諸字不可以定名。

## 符號

「單引號」注名，『雙引號』注法，《雙書名號》注模組，〈單書名號〉注品類。｜直｜注字，／左斜／注文。

#### 浮點

1.  長短不明者`一`；
2.  易美國《電機電子工程師學會標準第柒佰伍拾肆號》以補碼，零及無窮大不辨正負；
3.  内存若`符首...首尾....尾`
4.  如餘位全陰者，首二位
    - 陽陽：負一。（彼負二）
    - 陽陰：無窮大。（彼負零）
    - 陰陽：正一。（彼正二）
    - 陰陰：零。（彼正零）

$$x:s\ e_0 e_1 ... e_m\ t_0 t_1 t_2 ... t_n$$

$$
x=\begin{cases}
+2^{e_0 e_1 ... e_m - 2^m} \times {1.t_0 t_1 t_2 ... t_n}  && s=0\\
-2^{2^m - e_0 e_1 ... e_m} \times {1-0.0t_0 t_1 t_2 ... t_n} && s=1\\
+2^{-2^m+1} \times {0.t_0 t_1 t_2 ... t_n}  && e_0=e_1=...=s=0\\
-2^{-2^m+1} \times {1-0.t_0 t_1 t_2 ... t_n} && e_0=e_1=...=s=1\\
0 && s=e=t=0 \\
\infty && s=1,e=t=0 \\
\end{cases}
$$

- `實四` `實八`非必有。
- `複（一、二、四、八）`數：一實一虛，各如`實`數。

### 次

#### 諸列

以首字始、末字終。

```
列〈品〉甲置【首次】。
```

#### 諸物

物者多元合一。

```
物〈物名〉有〈品〉「元」(，〈品〉「元」)*。
```

#### 諸術

術者入多元而出一。

```
術「術名」({入|引}〈品〉「元」)*?出〈品〉。
  （所爲）
術成。
```

## 名數

### 名

「單引號」註變量，『雙引號』註常量，《雙書名號》註模組，〈單書名號〉註品類。

### 數

十進數：`〇一二三四五六七八九`\
十：`一〇`\
百：`一〇〇`\
千：`一〇〇〇`\
万/萬：`一〇〇〇〇`\
亿/億：`一〇〇〇〇〇〇〇〇`\
兆：`一〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇`\
十六進數：`〇一二三四五六七八九日月金木水火`。
小數：`整數．數位`

### 文

以『雙引號』注之。

## 語句

一以句點終。

### 定名

`〈品〉「名」置（式），［「名」置（式），...］`
更置不復書其類型。

### 定事

選擇：`若（條件甲）則...［否若（條件乙）則...］［否則...］也。`

循環：\
`若（條件）爲...云云。`\
`恆爲...云云。`\
`凡（列/物/類）中（元）...云云。`

跳轉：\
`至（定位）`: 限於一法之内。\
`止`：若至所「云云」後。\
`次`：若至所「云云」前。\
`得`：終其術，出一元。
