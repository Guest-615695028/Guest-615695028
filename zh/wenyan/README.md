# 統一文言編程語言

吾文言編程語言，一字以達意。

[彼](https://github.com/wenyan-lang/wiki/blob/master/Syntax-Cheatsheet.md)云：`吾有一數。曰三。名之曰「甲」。`\
我作：`整甲置三。`

彼云：`吾嘗觀「「算經」」之書。方悟「正弦」「餘弦」「圓周率」之義。`\
我作：`引《算經》得「正弦」「餘弦」「圓周率」。`\
註曰：`引《算經》得一`，「算經」諸法盡可用也。

彼云：`吾有一物。名之曰「甲」。其物如是。物之「「乙」」者。數曰三。物之「「丙」」者。言曰「「丁」」。是謂「甲」之物也。`\
我作：`物〈如是〉有數乙。言丙。物成。〈如是〉甲置「乙曰三，丙曰『丁』」。`

彼之法，蓋譯歐洲計製協會之編碼也，非中國人之法也。

## 關鍵字

```
數字：〇一二三四五六七八九十百千万亿兆京垓升降
（大寫）零壹貳叁肆伍陸柒捌玖拾佰仟萬億
品類：爻數整實複文列物類術入成引得
控制：若則否也恆爲云凡中至止次歸
運算：置用夫者之進退正負乘除模加減不大小等於在與抑或非
調試：書
```

右諸字不可以定名。

## 符號

「單引號」注名，『雙引號』注文，《雙書名號》注模組，〈單書名號〉注品類。
句點（。）終語句。

## 名數
### 爻
`〇`（`陰`）`一`（`陽`）二種。

### 數
十進數：`〇一二三四五六七八九`\
十：`一〇`\
百：`一〇〇`\
千：`一〇〇〇`\
万/萬：`一〇〇〇〇`\
亿/億：`一〇〇〇〇〇〇〇〇`\
兆：`一〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇〇`\
十六進數：`〇一二三四五六七八九日月金木水火`。
小數：`整數．數位`

### 文
以『雙引號』注之。

### 算
算符，依所先後
| 算符 | 元 | 註
|-|-|-
|`夫...者`|中|亦可用（圓括號）。
|`（列、物、類）之（名）`|前|列`「甲」之「乙」`決於`「乙」`，物類不然。
|`進 退 正 負 非`|後
|`乘 除 模`|自前而後
|`加 減`|自前而後
|`進 退`|自前而後|即左右移位，右移有正負補高，無正負補陰
|`（不）大於 （不）小於`|自前而後
|`（不）等於 （不）在`|自前而後
|`與`|自前而後|於整數按位
|`抑`|自前而後|同
|`或`|自前而後|同
|`用`|不結合|

例
| 彼云 | 我作 |
| --- | --- |
| `加一以二。` | `一加二` |
| `加一於二。` | `二加一` |
| `加一以二。乘其以三。` | `（一加二）乘三` |
| `除十以三。所餘幾何。` | `十模三` |
| `減七百五十六以四百三十三。名之曰「甲」。` | `數「甲」置七百五十六減四百三十三` |
| `夫「甲」「乙」中有陽乎。` | `甲與乙` |
| `夫「甲」「乙」中無陰乎。` | `甲或乙` |

## 語句
一以句點終。

### 定名
`〈品〉「名」置（式），［「名」置（式），...］`
更置不復書其類型。

### 定事
選擇：`若（條件甲）則...［否若（條件乙）則...］［否則...］也。`

循環：\
`若（條件）爲...云云。`\
`恆爲...云云。`\
`凡（列/物/類）中（元）...云云。`

跳轉：\
`至（定位）`: 限於一法之内。\
`止`：若至所「云云」後。\
`次`：若至所「云云」前。\
`得`：終其術，出一元。
| 彼云 | 我作 |
| --- | --- |
| `若三大於二者。乃...也。` | `若三大於二則...也。` |
| `若三不大於五者。乃...。若非。乃...也。` | `若三不大於五則...否則...也。` |
| `爲是百遍。...云云。` | `百遍...云云。` |
| `恆爲是。...云云。` | `恆爲...云云。` |
| `凡「天地」中之「人」。... 云云。` | `凡「天地」中「人」...云云。` |

### 注疏
`／＊注疏＊／`解於人耳目，編譯運行則去之。
| 彼云 | 我作 |
| --- | --- |
|`批曰。「「文氣淋灕。字句切實」」。`|`／＊文氣淋灕。字句切實＊／` |
|`注曰。「「文言備矣」」。`|`／＊文言備矣＊／` |
|`疏曰。「「居第一之位故稱初。以其陽爻故稱九」」。`|`／＊居第一之位故稱初。以其陽爻故稱九＊／` |
